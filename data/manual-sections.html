§1 Обыск

Обыск автомобиля: при аресте высокой опасности обязателен. Осмотрите салон фонариком, проверьте под сиденьями, торпедой, под капотом и в багажнике. Изымайте только ключевые улики. После обыска — эвакуатор на спецстоянку.

Обыск здания: при согласии подозреваемого или при ордере на арест и обыск. Будьте на чеку при входе в каждую комнату. Снимайте картины, проверяйте стены (глухой звук — возможная ниша). Изымайте улики, оружие, документы.

§2 Траффик-стоп повышенного риска (10-66)

Остановка транспорта с особо опасными преступниками: вооружённое нападение, розыск, погоня.

  • Дистанция до авто подозреваемого: 5–10 м; крузеры в шахматном порядке ЗА машиной.
  • Исключить перекрёстный огонь; патрульные в авто до приказа супервайзера.
  • Приказы — один человек (обычно пассажир ведущего крузера).
  • Руки вверх → заглушить двигатель → ключи в окно → выход спиной → «спешиться» → задержание высокой опасности.
  • Обыск авто — только после задержания всех подозреваемых.

§3 Действия штурма

Первый прибывший — красный офицер. Окружение → разведка → штурм.

  • Подход под BearCat; забаррикадированная дверь — таран BearCat.
  • Щиты — в начале штурмовой группы.
  • Без заложников: 2 предупреждения, те же действия без переговоров.
  • Tactical: тараны, дробовики со взламывающими зарядами, взрывпакеты, газ, flashbang, щиты.

§4 Ситуации с заложниками

Цель — сохранение максимального количества жизней. Классификации:

  • Бытовая — член семьи удерживает родных.
  • Психопат — резонанс, острые ощущения.
  • Безысходность — «живой щит» при побеге после задержания.
  • Террористы — профессионалы, политические цели; самая опасная категория.

До 3 предупреждений. При штурме — ликвидация стрелка; иногда допустимо касательное поражение через заложника (не жизненно важная зона).

§5 Переговоры

Переговорщик — обычная одежда + бронежилет (форма №303), без оружия.

  • Оценить вооружение и возможности террориста(ов).
  • Осмотреть здание, место угрозы, подходы, заряды ВВ.
  • При запросе транспорта — GPS-жучок, по возможности мало бензина.
  • После отдачи заложников и отъезда — преследование в unmarked (ЛТС), проверка на бомбу, задержание.

§6 Способы штурма помещений

  • Крючок — по обе стороны двери, flashbang, заход по стенам.
  • Крест — крестом по направлению стволов.
  • Угол — одна сторона входа, растекание по стенам (2 варианта).
  • Диагональ — вдоль одной стены с полным обзором.
  • Действия четвёрки — шеренга, спина к стене.
  • Две комнаты — зачистка первой, flashbang, «Крючок» во вторую.

§7 Разминирование (РБ)

ВУ с телефоном: глушилка → кусачки (вся связка) → телефон → деактивация.

ВУ с дистанционным управлением: глушилка → отвертка (корпус) → кусачки (провода) → снять детонатор.

ВУ с часами: корпус → индикаторная отвертка → перерезать «горящий» провод (или по температуре).

ВУ с кодовой панелью: как часы + дешифратор → код деактивации.

Мины, растяжки, гранаты — по регламенту таблицы РБ. Ликвидация ВВ — только на полигоне (заброшенный аэропорт).
