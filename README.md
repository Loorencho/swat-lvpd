# LVPD SWAT — Tactical Operations Portal

Набросок сайта подразделения SWAT в стиле [Call of Duty](https://www.callofduty.com/ru).

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
npx serve .
```

## Структура

| Раздел | Формат | Описание |
|--------|--------|----------|
| Hero | Главная карточка SWAT | Статус, командир, статистика |
| Состав | Карточки | Оперативники с званием и статусом |
| Арсенал | Карточки | Оружие, калибр, назначение |
| Справочник | Справочник сапёра | Типы СВУ, арсенал, RP-отыгровки |
| Новости | Карточки | Операции, набор, обновления |
| Тен-коды | Таблица | Радиообмен |
| Звания | Таблица | Система повышений |
| Расписание | Таблица | Тренировки на неделю |
| Мануал (краткий) | Пошаговый блок | Алгоритм операции + Use of Force |
| Ордера FBI | Список | Действующие директивы |
| Устав | Sidebar + контент | Навигация по разделам |
| Академия | Форма | Заявка на вступление |
| Internal Affairs | Форма | Жалобы на сотрудников |

## Стек

- HTML5 / CSS3 / Vanilla JS
- Шрифты: Bebas Neue, Rajdhani (Google Fonts)
- Без зависимостей
